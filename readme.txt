Yêu cầu:
- Node.js phiên bản 14.x trở lên
- MySQL

Cài đặt:
- Import bike-store.sql vào MySQL
- Đổi tên người dùng và mật khẩu phù hợp trong database.js
- Mở terminal trong thư mục đồ án và chạy
	npm install
- Để chạy web vào terminal gõ lệnh
	node app.js
- Đường dẫn mặc định của trang
	localhost:3000/


- Admin:
	admin
	123456
- Đường dẫn Admin Panel
	localhost:3000/admin

Github: https://github.com/tkthinh/web2-mySQL
